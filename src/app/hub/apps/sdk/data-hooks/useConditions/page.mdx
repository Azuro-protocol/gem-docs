# useConditions

The `useConditions` hook is used to fetch [`Conditions`](/knowledge-hub/how-azuro-works/components/conditions) of a specific game.

<Callout type="info">
Hook represents a logic wrapper over TanStack Query's `useQuery` hook. Explore [TanStack Query docs](https://tanstack.com/query/latest/docs/framework/react/reference/useQuery) to understand what data the hook returns.
</Callout>

## Usage

```ts
import { useConditions } from '@azuro-org/sdk'

const { data, isFetching, error } = useConditions(props)
```

## Props

```ts
{
  gameId: string | bigint | string[] | bigint[] // single game ID or array of game IDs
  onlyActiveOrStopped?: boolean // if true, returns only active or stopped conditions
  chainId?: ChainId
  query?: QueryParameterWithSelect<UseConditionsQueryFnData, TData> // useQuery params
}
```

```ts
type UseConditionsQueryFnData = ConditionDetailedData[]
```

```ts
type ChainId = 
  | 100     // Gnosis
  | 137     // Polygon
  | 80002   // Polygon Amoy
  | 88888   // Chiliz
  | 88882   // Chiliz Spicy
  | 8453    // Base
  | 84532   // Base Sepolia
```

<Callout type="warning">
  `gameId` property is not the same as `id`. Each game fetched using `useGames` hook contains the gameId:

  ```ts
  import { useGame } from '@azuro-org/sdk'

  const { data: games } = useGames()

  const gameId = games[0]?.gameId

  const { data } = useConditions({ gameId })
  ```
</Callout>

## Return Value

```ts
UseQueryResult<ConditionDetailedData[]>
```

```ts
import { type UseQueryResult } from '@tanstack/react-query'

type ConditionDetailedData = {
  conditionId: string
  gameId: string
  state: ConditionState
  /** true for stopped secondary conditions; consumed by useConditionState / useConditionsState */
  hidden?: boolean
  outcomes: Array<{
    outcomeId: string
    odds: string
  }>
  wonOutcomeIds?: string[]
  isExpressForbidden: boolean
  margin: string
  reinforcement: string
  virtualFund: string
  affiliate: string
  turnover: string
  wonOutcome?: string
  core: {
    address: string
    type: CoreType
    liquidityPoolAddress: string
  }
}

enum ConditionState {
  Created = 'Created',
  Active = 'Active',
  Stopped = 'Stopped',
  Resolved = 'Resolved',
  Canceled = 'Canceled',
  Paused = 'Paused'
}
```

## Query Options Helper

`getUseConditionsQueryOptions` lets you build query options outside a component — useful for prefetching, SSR, or composing queries.

```ts
import { getUseConditionsQueryOptions } from '@azuro-org/sdk'

const options = getUseConditionsQueryOptions({ ...props, chainId })
await queryClient.prefetchQuery(options)
```

```ts
type GetUseConditionsQueryOptionsProps = UseConditionsProps & {
  chainId: ChainId
}
```
