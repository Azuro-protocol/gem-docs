# useMaxBet

<Callout type="warning">
  **DEPRECATED**: This hook has been removed in SDK v7.0.0. Please use [`useBetCalculation`](/hub/apps/sdk/data-hooks/useBetCalculation) instead, which provides comprehensive bet calculation including maximum bet amount.
</Callout>

The `useMaxBet` hook is used for fetch maximum bet amount for provided user's selections.

## Usage

```ts
import { useMaxBet } from '@azuro-org/sdk'

const { data: maxBet, isFetching, error } = useMaxBet(props)
```

## Props

```ts
{
  selections: Selection[]
  chainId?: ChainId
  query?: QueryParameter<string>
}
```

```ts
type ChainId = 
  | 100     // Gnosis
  | 137     // Polygon
  | 80002   // Polygon Amoy
  | 88888   // Chiliz
  | 88882   // Chiliz Spicy
  | 8453    // Base
  | 84532   // Base Sepolia

type Selection = {
  outcomeId: string
  conditionId: string
}
```
