# 08/14 Multi Outcomes (Migration)

Azuro has carried out a successful migration to provide access to new features:

- Events with multiple outcomes. This will be the first step towards adding new sports;
- A new system for calculating rewards for frontend operators;
- New freebet management system.

The contracts migrated from 2.7.3 to 2.10.0 version.

<Callout type="alert">
  The new version of the protocol is incompatible with the previous version! Clients need to migrate and upgrade to the new version.
</Callout>

**Main changes:**

- New Core and Express(Combo) contracts;
- The methods working with native tokens moved from LP to ProxyFront;
- Conditions support more than 2 outcomes. No more need for additional regrouping (markets aggregation) on the client side.

<Callout type="info">
  You can find the detailed information about the release changes in <b>[07/21 Multi Outcomes (Preparation)](/releases/4-multi-outcomes-preparation)</b> release note.
</Callout>
