The `SocketProvider` facilitates the subscription functionality for real-time market updates during live games, specifically designed for Watch Hooks.

As for pre-match game's, live game's markets are constantly updated:

- Market can be stopped and unstopped.
- Outcomes odds can be changed.

#### Usage

Wrap your application in `SocketProvider`.

```tsx {1, 8, 10}
import { SocketProvider } from '@azuro-org/sdk'


function Providers(props: { children: React.ReactNode }) {
  const { children } = props

  return (
    <SocketProvider>
      {children}
    </SocketProvider>
  )
}
```
