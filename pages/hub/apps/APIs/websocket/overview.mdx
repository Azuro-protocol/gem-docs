# WebSocket API

The WebSocket API provides real-time updates for games and conditions, which is critical for live betting applications.

## Overview

The WebSocket API allows you to subscribe to real-time updates for games and conditions, ensuring that your application always has the most up-to-date information. This is particularly important for live betting, where odds and game states can change rapidly.

## Endpoints

### Development

- WebSocket URL: `wss://dev-streams.onchainfeed.org/v1/streams/feed`

### Production

- WebSocket URL: `wss://streams.onchainfeed.org/v1/streams/feed`
