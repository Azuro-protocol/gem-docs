Returns supported tokens for specific chain.

#### Usage

```ts
import { getDeBridgeSupportedTokens } from '@azuro-org/toolkit'

const chains = await getDeBridgeSupportedTokens(137)
```

#### Props
```ts
number
```

#### Return Value

```ts
Promise<Record<string, {
    address: string
    symbol: string
    decimals: number
    name: string
    logoURI: string
    tags: Array<string>
}> | null>
```
