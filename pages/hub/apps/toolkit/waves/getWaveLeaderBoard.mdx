Returns the [Wave Total Leaderboard](/hub/apps/guides/azuro-waves#fetch-total-leaderboard) data or [Wave Leaderboard for Specific Period](/hub/apps/guides/azuro-waves#fetch-leaderboard-for-specific-period).

#### Usage

```ts
import { getWaveLeaderBoard } from '@azuro-org/toolkit'

const periods = await getWaveLeaderBoard()
```

#### Props

```ts
{
  waveId?: WaveId
  account?: Address
  startsAt?: number
  chainId?: ChainId
}
```

```ts
type WaveId = number | 'active'

type ChainId =  100 | 137 | 80002 | 88888 | 88882
```

#### Return Value

```ts
Array<{
  position: number
  address: Address
  points: string
  bonusPoints: string | null
  totalMultipliedPoints: string
  bonusMultiplier: number | null
  level: number | null
  levelDescription: WaveLevelData | null
}> | null
```

```ts
type WaveLevelData = {
  level: number
  name: WaveLevelName
  boost: string
  pointsNeeded: string
  comment: string
}
```
