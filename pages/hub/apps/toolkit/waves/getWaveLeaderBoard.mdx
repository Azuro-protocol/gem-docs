Returns the [Wave Total Leaderboard](/hub/apps/guides/azuro-waves#fetch-total-leaderboard) or the [Wave Leaderboard for Specific Period](/hub/apps/guides/azuro-waves#fetch-leaderboard-for-specific-period).

#### Usage

```ts
import { getWaveLeaderBoard } from '@azuro-org/toolkit'

const periods = await getWaveLeaderBoard()
```

#### Props

```ts
{
  waveId?: WaveId
  account?: Address
  startsAt?: number
  chainId?: ChainId
}
```

```ts
type WaveId = number | 'active'

type ChainId = 
  | 100     // Gnosis
  | 137     // Polygon
  | 80002   // Polygon Amoy
  | 88888   // Chiliz
  | 88882   // Chiliz Spicy
  | 8453    // Base
  | 84532   // Base Sepolia
```

#### Return Value

```ts
Array<{
  position: number
  address: Address
  points: string
  bonusPoints: string | null
  totalMultipliedPoints: string
  bonusMultiplier: number | null
  level: number | null
  levelDescription: WaveLevelData | null
}> | null
```

```ts
type WaveLevelData = {
  level: number
  name: WaveLevelName
  boost: string
  pointsNeeded: string
  comment: string
}
```
