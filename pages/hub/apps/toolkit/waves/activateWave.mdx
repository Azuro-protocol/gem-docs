Used to [Activate User Levels](/hub/apps/guides/azuro-waves#activate-user-levels).

#### Usage

```ts
import { activateWave } from '@azuro-org/toolkit'

await activateWave({
  account: '0xkn123k...'
})
```

#### Props

```ts
{
  account: Address
  waveId?: WaveId
  chainId?: ChainId
}
```

```ts
type WaveId = number | 'active'

type ChainId = 
  | 100     // Gnosis
  | 137     // Polygon
  | 80002   // Polygon Amoy
  | 88888   // Chiliz
  | 88882   // Chiliz Spicy
  | 8453    // Base
  | 84532   // Base Sepolia
```

#### Return Value

```ts
Promise<void>
```
