import { Callout } from 'components'

Returns the user's cashback balance for the specified chain.

{/* <Callout type="info">
  You can find more information [Here](/hub/apps/guides/cashout/overview).
</Callout> */}

#### Usage

```ts
import { getCashbackBalance } from '@azuro-org/toolkit'


const { amount } = await getCashbackBalance({
  account: '0xlksnf...',
  affiliate: '0xlksnf...',
  chainId: 137,
})
```

#### Props

```ts
{
  account: Address
  affiliate: Address
  chainId?: ChainId
}
```

```ts
type ChainId = 100 | 137 | 80002 | 88888 | 88882 | 8453 | 84532

import { type Address } from 'viem'
```

#### Return Value

```ts
type GetCashbackBalance = {
    userAddress: Address;
    network: Environment;
    currency: string;
    updatedAt: string;
    amount: string;
} | null
```
