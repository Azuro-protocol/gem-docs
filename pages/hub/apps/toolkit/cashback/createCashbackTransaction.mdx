Returns a cashback transaction ready for submission on the blockchain.

{/* <Callout type="info">
  You can find more information [Here](/hub/apps/guides/cashout/overview).
</Callout> */}

#### Usage

```ts
import { createCashbackTransaction } from '@azuro-org/toolkit'


const tx = await createCashbackTransaction({
  account: '0xlksnf...',
  affiliate: '0xlksnf...',
  expiresAt: Math.floor(Date.now() / 1000) + 2000,
  chainId: 137,
})
```

#### Props

```ts
{
  account: Address
  affiliate: Address
  expiresAt: number
  chainId?: ChainId
}
```

```ts
type ChainId = 100 | 137 | 80002 | 88888 | 88882 | 8453 | 84532

import { type Address } from 'viem'
```

#### Return Value

```ts
type CashbackTransaction = {
  to: Address
  data: Hex
} | null
```

```ts
import { type Address, type Hex } from 'viem'
```
