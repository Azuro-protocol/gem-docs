Wrapper over [`useConditions`](/guides/sdk/data-hooks/useConditions) for groupping conditions by market.

#### Usage

```ts
import { useGameMarkets } from '@azuro-org/data'

const { loading, error, markets } = useGameMarkets(props)
```

#### Props

```ts
{
  gameId: string
  gameStatus: GameStatus // game status from `useGameStatus` hook or 'getGameStatus' function
  filter?: {
    outcomeIds?: string[]
  }
}

enum GameStatus {
  Created,
  Live,
  Resolved,
  Canceled,
  Paused,
  PendingResolution,
}
```

<Callout type="warning">
  `gameId` property is not the same as `id`. Each game fetched using `useGames` hook contains the gameId:

  ```ts
  import { useGame } from '@azuro-org/data'

  const { loading, error, games } = useGames()

  const gameId = games?.[0]?.gameId
  ```
</Callout>


#### Return Value

```ts
{
  loading: boolean
  error: Error | null
  markets: Market[]
}
```

```ts
type Selection = {
  conditionId: string | bigint
  outcomeId: string | bigint
}

type MarketOutcome = {
  selectionName: string
  odds: string
  lpAddress: string
  coreAddress: string
  status: ConditionStatus
  gameId: string | bigint
} & Selection

type Market = {
  name: string
  description: string
  outcomeRows: MarketOutcome[][]
}
```
