import { Callout } from 'components'

Hook is used to get the [Wave Total Leaderboard](/hub/apps/guides/azuro-waves#fetch-total-leaderboard) data or [Wave Leaderboard for Specific Period](/hub/apps/guides/azuro-waves#fetch-leaderboard-for-specific-period).

<Callout type="info">
  This hook is a wrapper around [getWaveLeaderBoard](/hub/apps/toolkit/getWaveLeaderBoard) function.
</Callout>

#### Usage

```ts
import { useWaveLeaderBoard } from '@azuro-org/sdk'

const { data } = useWaveLeaderBoard()
```

#### Props

```ts
{
  waveId?: WaveId
  account?: Address
  startsAt?: number // startsAt of specific period
  enabled?: boolean
}
```

```ts
type WaveId = number | 'active'
```

#### Return Value

```ts
UseQueryResult<WaveLeaderBoardItem[]>
```

```ts
import { type UseQueryResult } from '@tanstack/react-query'

type WaveLeaderBoardItem = {
  position: number
  address: Address
  points: string
  bonusPoints: string | null
  totalMultipliedPoints: string
  bonusMultiplier: number | null
  level: number | null
  levelDescription: WaveLevelData | null
}
```
