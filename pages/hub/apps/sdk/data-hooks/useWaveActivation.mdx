import { Callout } from 'components'

Hook is used to [Activate User Levels](/hub/apps/guides/azuro-waves#activate-user-levels).

<Callout type="info">
  This hook is a wrapper around [activateWave](/hub/apps/toolkit/activateWave) function.
</Callout>

#### Usage

```ts
import { useWaveActivation } from '@azuro-org/sdk'

const { activate } = useWaveActivation()

await activate()
```

#### Props

```ts
{
  account: Address
  waveId?: WaveId
}
```

```ts
type WaveId = number | 'active'
```

#### Return Value

```ts
{
  activate: UseMutateFunction
  activateAsync: UseMutateAsyncFunction
  isPending: boolean
}
```

```ts
import { UseMutateFunction, UseMutateAsyncFunction } from '@tanstack/react-query'
```
