# Live betting on Polygon & Gnosis mainnet

Azuro has launched live betting functionality on the Polygon and Gnosis mainnet with the max bet size limit of $10.

## To integrate into your dApp:

1. Upgrade the AzuroSDK to the latest version: https://github.com/Azuro-protocol/sdk

2. Or use the following guides:

- Live integration guide: https://gem.azuro.org/hub/apps/guides/advanced/live-tutorial
- How to connect live provider: https://gem.azuro.org/hub/apps/sdk/providers/live
- ClientCore deployment addresses: https://gem.azuro.org/contracts/deployment-addresses
- New endpoints (DataLiveFeed): https://gem.azuro.org/subgraph/overview
