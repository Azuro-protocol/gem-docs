# The Azuro Subgraph

Azuro uses multiple [subgraphs](https://thegraph.com/docs/about/introduction#what-the-graph-is) for indexing and
organizing data from the Azuro smart contracts. These subgraphs are hosted on The Graph hosted service and can be used
to query Azuro data.

[Github Repository](https://github.com/Azuro-protocol/Azuro-subgraphs/blob/main/api/README.md)
