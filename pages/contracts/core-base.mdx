# CoreBase

An abstract base contract for classic multi-outcome [Betting Engines](/knowledge-hub/how-azuro-works/components/betting-engines).

The CoreBase contract contains the main logic for managing [Conditions](/knowledge-hub/how-azuro-works/components/conditions), odds and
processing bets.
