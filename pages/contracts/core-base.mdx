# CoreBase

An abstract base contract for classic 2-outcome [Betting Engines](/concepts/protocol/betting-engines).

The CoreBase contract contains the main logic for managing [Conditions](/concepts/protocol/conditions) and
odds, processing bets, and calculating [Affiliate](/concepts/basic/affiliates)
[rewards](/concepts/basic/fees-and-rewards).
